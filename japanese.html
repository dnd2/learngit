第二課　これは家族の写真です

  1. これ/それ/あれ/どれ
    • それはなんですか。
    • これは家族の写真です。
    • 日本語の辞書はどれですか。
  2. この/その/あの/どの　+ 名
    • この建物は図書館です。
    • あの人は誰ですか。
    • 王さんの教科書はどの本ですか。
  3. 名は+誰のですか
    • このかばんは誰のですか。
    • そのかばんは佐藤さんのです。
  4. 何の+名ですか
    • それは何の辞書ですか。
    • これは英語の辞書です。
  5. 名+でした/では　ありませんでした
    • 昨日は金曜日でした。今日は土曜日です。
    • 昨日は休みではありませんでした。
  6. は(对比、强调)
    • 母は先生でした。今は主婦です。
    • この建物は図書館です。昔は食堂でした。

第四課　ギョーザは八個で五百円です

  1. 名1に 名2が [数词 + 助数词]　あります/います
    • 冷蔵庫にビールが２本あります。
    • 教室に学生が三人います。
    • 冷蔵庫に卵が五つあります。
    • 教室に学生が三十人います。
  2. 名に(は) 疑问词も　ありません/いません　　...没有(任何)... は表示强度或限定，可省略
    • 箱の中には何もありません。
    • 部屋には誰もいません。 房间里谁也不在
    • かばんの中に何もありません。
    • テーブルの上には何もありません。
  3. 名1に(は) 名2が [数词 + 助数词]も　あります/います　　　在...(竟然)有...， 数量词后的も强调数量之多
    • ダニエルさんの部屋にはパソコンが三台もあります。
    • 彼女の家には犬が五匹もいます。
    • 本棚の上には本が八十冊もあります。
    • 教室には一百人もいます。
  4. 名/[数词 + 助数词]しか　ありません/いません　　该句型强调数量至少，意为：只有...， 助词しか与否定表达方式连用表示限定。
    • 冷蔵庫には飲み物しかありません。
    • 自習室には学生が一人しかいません。
    • 駐車場に車が一台しかありません。
    • 図書館には三人しかいません。
    • 部屋には弟しかいません。
  5. で(合计)　　助词で与数量词或者表示范围的词连用时，表示累加、合计，意为：共...、加起来...
    • バナナは３つで一百五十円です。
    • りんごは２つで３百五十円です。
    • バナナは三本で一百五十円です。 三根香蕉150日元
    • 全部で一千五百円です。 总共1500日元
  6. 名を　ください 意为：请给我...，多用在购物、用餐等情景中，"を　ください"带有轻微命令语气，一般不对长辈或上级使用
    • りんごを４つください。
    • コーヒーとケーキをください。
    • りんごを十個ください。
    • ごご電話をください。　下午请给我打电话
  • せん　はっぴゃくえん　　1,800
  • いちまん　ろっぴゃく　さんじゅうえん 10,630
  • さんまん　さんぜん　ななひゃく　よんじゅう　にえん 33,742
  • じゅうまん　ろくせん　ななひゃく　よんじゅう　にえん 106,742
  • にひゃく　ごじゅう　はちまん　ひゃく　ろくじゅう　はちえん 2,580,168

第五課　今朝　七時に　おきました

  1. 毎日七時に学校へ行きます。

  2. 毎日学校へ行きます。

    • 今日は学校へ行きません。
    • 昨日は学校へ行きました。
    • おとといは学校へ行きませんでした。

  3. わたしは七時半に起きます。

    • 鈴木さんは何時に起きますか。
    • ごご四時に来ます。
    • わたしは昨日学校へ行きませんでした。

  4. ダニエルさんは八時に学校へ行きます。

    • ジョンさんはどこへ行きましたか。
    • 寮へ行きました。

  5. 朝ご飯を食べます。

    • テレビを見ます。
    • 新聞を読みます。
    • 午前中何をしますか。 上午做什么?
    • にほんごを勉強します。

  6. わたしはスポーツはあまりしません。 は除提示主题、表示对比外，还可起到强调或限定作用

    • 冷蔵庫にはビールはありません。　　代替が
    • わたしはスポーツはあまりしません。代替を
    • 午後には会議があります。 与に连用
    • 僕はイギリスへは行きません。 与へ连用

  • ～たち 结尾词，常接在表人称的名词，人称代词后，表示复数，相当于...们

    • 私たち、学生たち
    • 除たち可表人的复数外，～がた也可表示复数，表复数时是一种尊称，包含尊敬的语气
      • 先生がた(老师们)、奥様がた(夫人们)

  • ～ごろ 接表时间词语后，如～年、～月、～曜日、～日、～時、～分表示大概的时间范围，相当于...左右，...前后，后面不加に

    • 毎朝六時ごろ起きます。
    • 毎日七時半ごろ会社へ行きます。

  • ～そして　接续词，可连接句子，短语。

    • 表示并列、追加等，相当于“而，又”等
      • 姉は大学の先生です。そして、妹は会社員です。
      • 王さんは図書館にいます。そして、キムさんは自習室にいます。
    • 表示继前项事情后，凡是后项的事情，相当于"然后，之后"
      • 日本語の勉強をします。そして、テレビを見ます。
      • 今朝、朝ご飯を食べました。そして、新聞を読みました。

  • あまり　副词，一般跟否定形式呼应使用，表示频率，程度不高，"不怎么...,不太..."

    • 朝ご飯はあまり食べません。
    • 佐藤さんはあまりテニスをしません。

  • くらい/ぐらい 接表示数量的词后，表示大概的数量。2者意思相同

    • 教室には学生が二十人くらいいます。
    • 今朝、十分ぐらい新聞を見ました。

第六課　週末は何をしますか

  1. 教室で日本語を勉強します。　助词で表示动作进行的场所
    • レストランで昼のご飯を食べました。
    • とごで本を読みますか。
      • 図書館で本をよみます。
  2. アメリカに行きます。
    1. に表示动作，行为的目的是，常与行きます、帰ります、来ます、戻ります等包含移动的次一起使用
      • 日本にいきます。
      • 家に帰ります。
      • 会社に戻ります。　回公司
    2. 表示行为的目的,常与行きます、帰ります等连用构成～へ ～に　行きます/帰ります/来ます/戻ります的句型，助词に前常接表示行为目的地的名词或动词ます形去掉ます后的形式。 (～へ ～に ～ます)
      • 名词 + に
        • 公園へ花見に行きます。
        • デパートへ買い物に行きます。
        • 図書館へ本を借りに行きます。
      • 动词 + に
        • 池袋へ映画を見に行きます。
        • わたしは写真を取りに来ました。我是来取照片的
        • 公園へ何をしに行きますか。　你去公园做什么
          • 散歩に行きます。 去散步

第七課　日本の秋は涼しいです

  1. 日本の秋は涼しいです。
    • 東京の生活は便利です。
  2. 今日は暑くないです。
    • 教室は静かではありません。
  3. これは新しいパソコンです。
    • ここは有名な神社です。
  4. 王さんはもう帰りました。
  5. あの店の料理は安いですが、あまりおいしくないです。
  6. 教室に誰かいますか。
  7. 彼女に電話をしました。

  • この小説は面白くないです。

  • 東京の冬は寒くありません。

  • そのかばんはあまり丈夫ではありません。

  • 鈴木さんは面白いです。

  • 北海道はきれいな所です。

  • 奈良は静かな町です。

  • もう　+ 動ました　表示已经发生的事情或过去的状态

    • 木村さんはもう学校に行きました。
    • 田中さんはもう家に帰りました。
    • 王さんはもう日本にいきました。

  • ～が、表转折，意思是...但是...

    • 四川料理はおいしいですが、とても辛いです。
    • 今日は安いですが、明日は学校へ行きます。
    • その店は狭いですが、商品がたくさんあります。

  • 疑问词+か　助词か接在だれ、どこ、なに等疑问词后表示不确定

    • 部屋に誰かいますか。
    • どこかへ遊びに行きましょう。
    • キムさんは何か言いましたか。

  • に表示动作的对象、接受者以及动作的着落点等

    1. 表示动作的对象、接受者
      • 母に電話をしました。
      • 妹に英語を教えます。
      • 王さんに会いました。
    2. 表示动作着落点
      • シャンハイに着きました。
      • バスに乗ります。
      • 本をかばんに入れます。

第十課　電子辞書は機能が大きくなりました

    • 息子は教師になりました。
    • 彼女はきれいになりました。
    • 彼は以前より真面目になりました。
  2. イ形 + く　なります
    • 物価は高くなりました。
    • だんだん暖かくなりました。
    • うちのワンちゃんは大きくなりました。
  3. 名/形 + に します
    • 会議室を禁煙にします。
    • 部屋をきれいにします。
  4. イ形 + く します
    • テレビの音を大きくします。
    • 文章を短くします。
  5. 名1で 名2が 一番 ~です
    • クラスで王さんが一番若いです。
  6. 名で 疑问词が 一番　~ですか
    • スポーツの中で何が一番好きですか。
  7. ~は　~が　~です
    • 子の機械は操作が簡単です。
  • すみません、電子辞書売り場はどこですか。
  • こちらは今月新発売の最新機種でございます。

  • こちらの新しい３機種の特徴はそれぞれ何ですか。

第十一課　牛丼の作り方を教えてください

  1. 書いて/見て/して/来て
    • 買う ｰｰ>　かって
    • 持つ ｰｰ>　持って　拿，持有 荷物を持つ　拿东西
    • 帰る ｰｰ>　帰って
    • 呼ぶ ｰｰ>　呼んで　叫，喊
    • 読む ｰｰ>　読んで
    • 死ぬ ｰｰ>　死んで
    • 書く ｰｰ>　書いて
    • 泳ぐ ｰｰ>　泳いで
    • 話す ｰｰ>　話して　说
    • 食べる ｰｰ>　食べて
    • 起きる ｰｰ>　起きて
    • 開ける ｰｰ>　開けて
    • する ｰｰ>　して
    • 来る ｰｰ>　来て
  2. 朝ご飯を食べて、学校へ行きます。　动词て形可连接2个或2个以上的动词，表示按照动作先后顺序发生，意为：...之后
    • 昨日図書館へ行って、本を借りました。
    • スーパーへ行って、買い物をして、家へ帰ります。去超市买完东西后回家
      • 窓を閉めて、寝ます。「閉める」
      • 調味料を入れて、煮ます。「入れる」装进，放入
      • デパートへ行って、買い物をします。
      • 家に帰って、宿題をします。
  3. シャワーを浴びてから、寝ます。 动词て形 + から 表示动作按时间先后顺序相继发生，意为：在...之后
    • 朝ご飯を食べてから、出かけます。　吃完早饭后出门
    • 会社に行ってから、電話をします。
    • 宿題をしてから、テレビを見ます。 做完作业后再看电视
      • 部屋を掃除してから、出かけます。
      • 日本へ来てから、日本語を勉強します。
      • 宿題をしてから、テレビをみます。
      • ジョギングしてから、シャワーを浴びます。
      • 朝ご飯を食べてから、新聞を読みます。
      • 玉ねぎが柔らかくなってから、牛肉を入れます。
  4. ダニエルさんは今新聞を読んでいます。　动词て形 + います表示正在进行的动作或经常、反复进行的动作
    • ドアが開いています。
    • 週に二回日本語教室に通っています。「通う」每周去2次日语培训班学习
    • 动词て形 + います还可表示状态的持续以及动作结果的存续
      • 彼女はシャンハイに住んでいます。　「住む」她住在上海
      • わたしがドアを開けました。今ドアは開いています。是我把门打开的，现在门是开着的

注意的句型

  1. 王さん、週末はいつも何をしますか。　　小王，(你)周末都在干什么啊
    • ダニエルさんは週末に何をしますか。丹尼尔(你)周末一般做什么啊
  2. これから池袋へ映画を見に行きます。王さんも一緒に行きませんか。　(我)一会儿去池袋看电影，小王(你)要不要一起去啊
  3. これから食堂へ昼ご飯を食べに行きます。佐藤さんも一緒にいきませんか。
    • いいですね。一緒に行きましょう。
